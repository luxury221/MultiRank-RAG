# 实验与指标说明

## 对比组

- G0：原始候选顺序。
- G1：仅语义相似度。
- G2：语义相似度 + 查询感知 PPR。
- G3：语义相似度 + 查询感知 PPR + Bridge。

## 指标

- `Recall@K`：正确证据是否出现在前 K 个结果中。
- `MRR`：第一个正确证据越靠前分越高。
- `nDCG@5`：衡量 Top-5 排序质量。
- `Evidence Hit Rate`：Top-5 是否命中正确证据。
- `Modality Hit Rate`：Top-5 是否命中正确证据模态。
- `Citation Accuracy`：Top-5 是否覆盖正确页码。
- `Rerank Time`：G2/G3 的轻量重排序耗时。

## 权重实验

建议至少试三组：

```text
A: lambda_s=0.7, lambda_p=0.2,  lambda_b=0.1
B: lambda_s=0.6, lambda_p=0.25, lambda_b=0.15
C: lambda_s=0.5, lambda_p=0.25, lambda_b=0.25
```

选择标准：

- 总体指标是否提升。
- 表格题、图表题、图文一致性题是否提升。
- 文本事实题是否被明显拖累。
- 重排序耗时是否可接受。
